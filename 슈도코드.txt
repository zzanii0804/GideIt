메인페이지
START

1. 메인 컨테이너 div 생성
    - 이 컨테이너는 모든 섹션(header, content, footer)을 포함.

2. 헤더 섹션
    - 겹쳐진 요소들을 포함하는 헤더 섹션 생성.
        - 문자열을 입력시킬 수 있는 검색바가 있는 섹션 추가
        - 메인페이지로 연결되는 로고 이미지 추가.

3. 메인 콘텐츠 섹션
    - 카테고리 메인 콘텐츠 컨테이너 생성.
    
    - 이미지와 레이블이 포함된 카테고리 목록 생성:
        - "모니터" - 모니터 이미지 및 레이블 추가.
        - "블루투스이어폰" - 블루투스이어폰 이미지 및 레이블 추가.
        - "보조배터리" - 보조배터리 이미지 및 레이블 추가.
        - "태블릿" - 태블릿 이미지 및 레이블 추가.
        - "TV" - TV 이미지 및 레이블 추가.
        - "노트북" - 노트북 이미지 및 레이블 추가.
        - 각 아이템에 링크 추가
          (카테고리의 제품 이미지 클릭 시 해당하는 제품의 구매 가이드 페이지로 이동).

4. 푸터 섹션
    - 팀 정보를 포함하는 푸터 섹션 생성.
    
    - 팀 정보 추가:
        - "팀명" - "Guide It" 텍스트 추가.
        - "팀원" - "권혁찬, 정현우" 이름 추가.
        - "주소" - "서울특별시 서대문구 가좌로 134, 명지전문대학교" 주소 추가.

END
